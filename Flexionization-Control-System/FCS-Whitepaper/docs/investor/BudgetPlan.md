# 12–18 Month Budget Plan

This document outlines the projected budget required to bring the FCS project from a validated prototype to a commercially ready control platform.  
Planning horizon: **12–18 months**.  
Target raise: **$300,000 – $500,000**.

---

## 1. Budget Overview (12 Months)

| Category                         | Estimated Cost Range     | Notes                                      |
|----------------------------------|---------------------------|--------------------------------------------|
| **Engineering & R&D**           | $120,000 – $200,000       | Core team salaries, math, integration      |
| **Hardware & Testing**          | $80,000 – $150,000        | UAVs, sensors, robotics platforms          |
| **Documentation & Developer Tools** | $40,000 – $70,000     | Docs, tutorials, CI/CD, developer support  |
| **Operations & Partnerships**   | $20,000 – $40,000         | Integrations, travel, certification prep   |
| **Contingency (10%)**           | $30,000 – $50,000         | Safety buffer for unexpected expenses      |

Total estimated cost for 12 months: **$290,000 – $510,000**

(Aligned with the target raise.)

---

## 2. Quarterly Breakdown (12 Months)

### **Q1 — Core Foundation**
- Finalize Whitepaper v1.0  
- Stabilize SDK architecture  
- Expand examples (servo, UAV axis, robotic joint)  
- Documentation improvements  

**Estimated cost:** $60,000 – $90,000

---

### **Q2 — Integration & Validation**
- PX4/ArduPilot simulation integration  
- Hardware test bench (STM32 + IMU + servo)  
- Benchmarking against PID  
- Release SDK v1.2  

**Estimated cost:** $70,000 – $110,000

---

### **Q3 — Scaling & Productization**
- Plug-and-play MCU modules  
- ROS2 integration package  
- Automatic tuning tools  
- Release SDK v1.3  

**Estimated cost:** $80,000 – $120,000

---

### **Q4 — Commercial Readiness**
- FCS v1.0 stable release  
- Safety/fallback layers  
- Pilot deployments with partners  
- Licensing model preparation  

**Estimated cost:** $80,000 – $120,000
